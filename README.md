# 🔐 Vigenère Cipher in C

A simple implementation of the Vigenère Cipher encryption algorithm using the C programming language.

## 📌 Features

* Encrypts plaintext using a keyword
* Preserves uppercase and lowercase letters
* Ignores non-alphabet characters
* Saves encrypted output to a log file

## ⚙️ How to Run

### Compile

gcc main.c -o main

### Run

./main

## 📝 Example

Enter the plaintext: HELLO
Enter the key: KEY
Ciphertext: RIJVS

## 📁 Output

Encrypted data is stored in:
vigenere_log.txt
